# sudoku_solver
# 🧩 Glass Sudoku Solver – Streamlit App


## ✨ Features

- ✅ **9×9 Input Grid** – Easy‑to‑use number inputs (0 for empty cells).  
- ✅ **Backtracking Solver** – Efficient recursion with pruning for speed.  
- ✅ **Full Validation** – Checks row, column, and 3×3 sub‑grid for conflicts before solving.  
- ✅ **Glassmorphism UI** – Semi‑transparent cards, blur effects, and soft shadows for a premium look.  
- ✅ **Sample Puzzle** – Pre‑loaded example from the task description.  
- ✅ **Clear & Reset** – One‑click clear or reload the sample puzzle.  
- ✅ **Instant Feedback** – Success or error messages with Streamlit alerts.
